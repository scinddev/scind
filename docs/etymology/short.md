# Why "Scind"?

**Scind** comes from the Latin verb *scindere*: to cut, to split, to cleave apart.

It is the root behind English words like *scissors*, *rescind*, and *schism* — each carrying the idea of a decisive separation.

Scind does exactly this for Docker Compose environments. It takes a multi-application stack and cleanly splits it into isolated workspaces — independent copies that share nothing with each other. Your development environment, your code review environment, and your control baseline all run simultaneously on the same host, each one cut off from the others.

The name is deliberately short and imperative. Five letters. One syllable. A sharp, clean cut — which is precisely what the tool delivers.
